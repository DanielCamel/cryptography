# Генератор Геффе

Простая реализация генератора Геффе - криптографического алгоритма на основе линейных регистров сдвига с обратной связью (LFSR).

## Описание

Генератор Геффе использует три LFSR (LFSR-1, LFSR-3 и управляющий LFSR-2) для создания псевдослучайной последовательности:
- **R0**: полином x⁴ + x³ + 1
- **R1**: полином x⁴ + x + 1  
- **CTRL**: полином x⁴ + x³ + x² + x + 1

# RSA (Rivest-Shamir-Adleman)

Реализация криптосистемы RSA для шифрования и расшифрования данных.

## Описание

RSA - асимметричный криптографический алгоритм, основанный на сложности факторизации больших чисел:

- **p, q**: два случайных простых числа в диапазоне 128-256 (16-битный модуль)
- **n = p × q**: RSA-модуль
- **φ(n) = (p-1)(q-1)**: функция Эйлера
- **e**: открытая экспонента, взаимно простая с φ(n) (выбирается из [3, 5, 7, 11, 13, 17, 65537])
- **d**: закрытая экспонента, вычисляется как мультипликативно обратная к e по модулю φ(n) с использованием расширенного алгоритма Евклида

## Алгоритм

### Генерация ключей
1. Выбираются два простых числа p и q
2. Вычисляется n = p × q и φ(n) = (p-1)(q-1)
3. Выбирается открытая экспонента e, где gcd(e, φ(n)) = 1
4. Находится закрытая экспонента d ≡ e⁻¹ (mod φ(n))

### Шифрование
```
c ≡ m^e (mod n)
```
где m - открытый текст в виде целого числа

### Расшифрование
```
m ≡ c^d (mod n)
```
где c - шифртекст

## Использование

```bash
python RSA.py
```

Введите открытый текст и шифртекст в двоичном виде (только цифры 0 и 1).

Программа вычислит и выведет:
- Выбранные простые числа p и q
- RSA-модуль n
- Открытый ключ (e, n)
- Закрытый ключ (d, n)
- Результаты шифрования и расшифрования
